## 3. create-react-app

이름에서 알 수 있듯이, 리액트 앱을 만들어주는 도구입니다.

### 설치하기

```bash
# npm을 통해 설치
npm install -g create-react-app

# yarn을 통해 설치
yarn global add create-react-app
```

### 사용하기

```bash
create-react-app [Project Name]
```
